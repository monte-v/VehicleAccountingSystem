# Учет транспортных средств

Графическое приложение для учета транспортных средств, разработанное с использованием **Qt Widgets**, паттерна **MVC (Model–View–Controller)** и Qt **5.x**.

## Возможности

- Добавление транспортного средства
- Редактирование записей
- Удаление записей
- Сохранение списка в JSON-файл
- Загрузка списка из JSON-файла
- Сортировка записей по столбцам
- Фильтрация по типу транспортного средства

## Структура записи

Каждое транспортное средство содержит следующие поля:

- Уникальный идентификатор (числовой)
- Тип транспортного средства
- Марка
- Модель
- Год выпуска
- Вес
